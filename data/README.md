# Financial Distress Capstone Project

This repository contains the dataset, code, and output files used for the QM640 Data Analytics Capstone interim report.

## Project Title
Explainable AI for Early Warning Assessment of Financial Distress and Financial Statement Red Flags

## Contents

- `data/` - Raw financial dataset
- `notebooks/` - Python notebook used for EDA and preliminary modelling
  

## Data Source
Financial data was extracted from Screener: https://www.screener.in/

## Note on Distress Label
The distress label used in the interim report is based on a simplified rule-based approach using selected financial ratios. A more robust distress identification framework will be developed in the final report.

## Models Used
- Logistic Regression
- Random Forest
